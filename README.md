# Invoices App
It is an application for simple managing invoices, created in Angular.

## Author
- Dariusz Okonski

## Technologies
- Angular
- TypeScript
- Bootstrap

## Installing
- Node is needed to be installed on a local machine
- Download repository
- Open terminal in repository main folder
- Type: npm install
- Type: npm run start
